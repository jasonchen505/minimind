# MiniMind LLM入门项目复现学习笔记

## 一、项目概述

### 1.1 项目简介
MiniMind是一个从零开始训练小型语言模型(LLM)的完整开源项目，目标是让普通个人GPU也能快速完成训练与复现。

**核心特点：**
- 模型规模：约64M参数（GPT-3的1/2700）
- 训练成本：约3块钱人民币，2小时训练时间
- 完整流程：覆盖预训练、SFT、LoRA、RLHF(DPO)、RLAIF(PPO/GRPO)等全流程
- 纯PyTorch实现：所有核心算法从0实现，不依赖第三方高层抽象

### 1.2 项目结构
```
minimind/
├── model/              # 模型定义
│   ├── model_minimind.py   # MiniMind模型架构
│   └── model_lora.py       # LoRA实现
├── trainer/            # 训练脚本
│   ├── train_pretrain.py   # 预训练
│   ├── train_full_sft.py   # SFT微调
│   ├── train_lora.py       # LoRA微调
│   ├── train_dpo.py        # DPO强化学习
│   ├── train_ppo.py        # PPO强化学习
│   └── train_grpo.py       # GRPO强化学习
├── dataset/            # 数据集处理
│   └── lm_dataset.py       # 数据集类定义
├── scripts/            # 工具脚本
│   ├── eval_llm.py         # 模型评估
│   └── web_demo.py         # WebUI演示
└── dataset/            # 数据集文件
    ├── pretrain_t2t_mini.jsonl  # 预训练数据(1.2GB)
    └── sft_t2t_mini.jsonl       # SFT数据(1.6GB)
```

## 二、环境配置

### 2.1 硬件环境
- **GPU**: NVIDIA GeForce RTX 3090 (24GB) × 8
- **CUDA**: 12.1
- **系统**: Ubuntu Linux

### 2.2 软件环境配置

使用`uv`创建虚拟环境：
```bash
# 创建虚拟环境
uv venv /home/chenyizhou/minimind/.venv --python 3.12

# 激活环境
source /home/chenyizhou/minimind/.venv/bin/activate

# 安装PyTorch (CUDA 12.1)
uv pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121

# 安装项目依赖
uv pip install -r requirements.txt
```

### 2.3 依赖包说明
主要依赖包括：
- `torch==2.5.1+cu121`: PyTorch深度学习框架
- `transformers==4.57.6`: HuggingFace Transformers库
- `datasets`: 数据集加载库
- `wandb`/`swanlab`: 训练可视化工具
- `tiktoken`: Tokenizer

## 三、数据准备

### 3.1 数据集下载
从ModelScope下载数据集：
```bash
# 使用wget下载
wget --no-check-certificate "https://www.modelscope.cn/api/v1/datasets/gongjy/minimind_dataset/repo?Revision=master&FilePath=pretrain_t2t_mini.jsonl" -O ./dataset/pretrain_t2t_mini.jsonl

wget --no-check-certificate "https://www.modelscope.cn/api/v1/datasets/gongjy/minimind_dataset/repo?Revision=master&FilePath=sft_t2t_mini.jsonl" -O ./dataset/sft_t2t_mini.jsonl
```

### 3.2 数据格式说明

**预训练数据格式（pretrain_t2t_mini.jsonl）：**
```json
{"text": "如何才能摆脱拖延症？治愈拖延症并不容易，但以下建议可能有所帮助。"}
{"text": "Transformer 通过自注意力机制建模上下文关系，是现代大语言模型的重要基础结构。"}
```

**SFT数据格式（sft_t2t_mini.jsonl）：**
```json
{
    "conversations": [
        {"role": "user", "content": "你好"},
        {"role": "assistant", "content": "你好！"}
    ]
}
```

### 3.3 数据集大小
- `pretrain_t2t_mini.jsonl`: 1.2GB，约127万条数据
- `sft_t2t_mini.jsonl`: 1.6GB，约90万条数据

## 四、学习心得

### 4.1 项目设计理念
1. **极简原则**：用最少的代码实现完整功能
2. **从零实现**：不依赖第三方高层抽象，便于理解底层原理
3. **低成本复现**：让普通开发者也能训练自己的LLM

### 4.2 关键技术点
- **RoPE旋转位置编码**：相对位置编码，支持长序列外推
- **RMSNorm**：相比LayerNorm更稳定
- **SwiGLU激活函数**：比ReLU效果更好
- **GQA(Grouped Query Attention)**：平衡效果和效率

### 4.3 环境配置经验
1. 使用`uv`管理虚拟环境更高效
2. ModelScope下载数据比HuggingFace在国内更稳定
3. RTX 3090的24GB显存足够训练64M参数模型
